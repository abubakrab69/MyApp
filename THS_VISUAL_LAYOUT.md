## Interactive Elements

### **Image Navigation**
```
Main Image
    ↓
[Prev Button] ← Click to go back
[Next Button] ← Click to go next
    ↓
OR
    ↓
Click Any Thumbnail → Jumps to that image
    ↓
Shows in Counter: "3 / 11"
```

### **Card Interactions**
```
Hover Over Card:
  Background color changes (lighter)
  Border becomes brighter
  Shadow gets stronger
  Slight lift animation (translateY)
```

### **Button Interactions**
```
Hover Over Button:
  Background gradient brightens
  Border becomes more visible
  Text becomes whiter
  Button lifts up slightly
  Shadow appears/strengthens
```

### **Thumbnail Interactions**
```
Hover Over Thumbnail:
  Image zooms slightly (1.05x)
  Border gets brighter
  
Click Thumbnail:
  Becomes active (pink border)
  Main image switches instantly
  Counter updates
  Animation plays
```

## Color Coding

### **Section Titles**
```
📱 = Blue/Purple (Project Overview)
💻 = Blue (Technology Stack)
⭐ = Gold/Yellow (Key Features)
✨ = Pink/Magenta (Highlights)
🛠️ = Orange (Development)
```

### **Card Styling**
```
Background:  rgba(255, 255, 255, 0.03)  [Very transparent]
Border:      rgba(255, 255, 255, 0.1)   [Subtle white border]
Hover BG:    rgba(255, 255, 255, 0.05)  [Slightly lighter]
Hover Border: rgba(102, 126, 234, 0.4)  [Blue tint]
```

### **Badges**
```
Tech Badges:      Purple/Blue gradient
Feature Badges:   Pink/Magenta
Stat Boxes:       Purple with gradient text
Highlight Items:  Pink left border
```

## Animation Timeline

### **Page Load (0-1.2s)**
```
T=0ms:     Start fade in from bottom
T=200ms:   Title fades in
T=300ms:   Title underline expands
T=400ms:   Image section fades in
T=600ms:   Details section fades in
T=800ms:   Cards start appearing
T=1000ms:  All animations complete
T=1200ms:  Page fully loaded
```

### **Image Switch (0-0.5s)**
```
T=0ms:     Click thumbnail/button
T=100ms:   Old image starts fading
T=250ms:   New image fades in
T=500ms:   Transition complete
```

### **Hover Effects (0-0.3s)**
```
T=0ms:     Mouse hover detected
T=50ms:    Border color changes
T=100ms:   Background lightens
T=150ms:   Card lifts up
T=300ms:   All effects applied
```

## Layout Specifications

### **Spacing**
```
Header to Content:  60px
Content Gap:        50px (desktop) / 30px (tablet) / 20px (mobile)
Card Padding:       28px (desktop) / 20px (tablet) / 16px (mobile)
Section Gap:        24px
Image to Buttons:   20px
```

### **Sizes**
```
Main Title:         2.5rem - 4rem (responsive)
Section Title:      1.4rem
Subtitle:          1.3rem
Body Text:         0.95rem
Small Text:        0.85rem
```

### **Thumbnails**
```
Desktop:    60px (width/height)
Tablet:     50px
Mobile:     45px
Aspect:     9:16 (phone ratio)
```
